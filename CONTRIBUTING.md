# Contributing to Questline

Thank you for your interest in contributing to Questline.

## How to Contribute
1. Fork the repository
2. Create a new branch for each issue
3. Follow the instructions mentioned in the issue
4. Keep changes limited to the specified folders
5. Commit with clear messages
6. Open a Pull Request

## Rules
- All custom code must be inside Assets/_Project
- Do not modify files outside the allowed scope
- Follow Unity version 6.3 LTS (6000.3.2f1)
